# Question Answering Chatbot using GenAI

This project is a simple Question Answering Chatbot built using Hugging Face Transformers and Gradio. The chatbot takes a context and a question from the user, then generates an answer using a pre-trained NLP model.

## Technologies Used
- Python
- Hugging Face Transformers
- Gradio
- NLP (Natural Language Processing)

## Features
- Simple and interactive chatbot interface
- Question answering using transformer models
- Easy to understand and beginner friendly
- Web interface using Gradio

## How It Works
1. User enters a context paragraph
2. User asks a question related to the context
3. The model processes the input
4. The chatbot returns the most relevant answer

## Installation

Install the required libraries:

```bash
pip install transformers gradio torch
```

## Run the Project

Run the notebook or use:

```bash
python app.py
```
